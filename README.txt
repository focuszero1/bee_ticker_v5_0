
Bee Ticker v5.0 Project Structure

To run:
1. Install dependencies:
    pip install feedparser

2. Run the app:
    python main.py

Modules:
- main.py     → app launcher + main logic
- feeds.py    → handles pulling news feeds
- storage.py  → manages saved articles file
- gui.py      → tkinter + ttk UI code
- themes.py   → dark/light mode style functions
- utils.py    → helper functions

You can safely start filling in code in each module.

This structure matches your working Bee Ticker v4.0 functionality.
